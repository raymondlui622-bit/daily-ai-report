# Daily AI Intelligence Report

Automated Python script that generates and emails a daily AI briefing.

## Features
- GitHub Trending scraping
- Hacker News scraping
- AI news via RSS
- Claude-powered summarization
- Email delivery

## Setup

1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt

3. Create `.env` file:
   - ANTHROPIC_API_KEY
   - Gmail credentials

4. Run:
   python3 main.py --no-email
   python3 main.py
